## 🔧 Getting Started

To get a local copy up and running, please follow these simple steps.

### Prerequisites

-   Python 3.x, pip
-   Node.js & npm (optional for frontend tools)
-   Android Studio or Flutter SDK

### Installation

1.  **Clone the repo**
    ```sh
    git clone [https://github.com/PunyaKSirohi/FocusFlow.git](https://github.com/PunyaKSirohi/FocusFlow.git)
    ```
2.  **Backend Setup**
    ```sh
    cd FocusFlow/backend
    pip install -r requirements.txt
    python manage.py runserver
    ```
3.  **Frontend Setup (Example for Flutter)**
    ```sh
    cd FocusFlow/frontend
    flutter pub get
    flutter run
    ```
